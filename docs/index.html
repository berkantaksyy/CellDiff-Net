ML-Based Identification of Stem Cell Differentiation States from Microscopy Images

Combining representation learning, segmentation, and biological interpretability to identify stem cell differentiation states from label-free microscopy.

Berkant Aksyy
GitHub Slides Paper (Coming Soon) References

Abstract

Identifying stem cell differentiation states is a fundamental challenge in cell biology, typically addressed through molecular markers or time-consuming expert visual inspection. This project investigates whether deep learning models can learn to classify or score differentiation states directly from microscopy images, in a label-free and reproducible manner. The current phase focuses on a systematic literature review of existing approaches — spanning cell segmentation, representation learning, and self-supervised methods — to define a robust pipeline for future experimental validation. Model training and evaluation code will be added as the project matures.

Proposed Pipeline

Raw Microscopy Images
Brightfield, phase-contrast, or fluorescence input
Preprocessing & QC
Illumination correction, normalization, plate/batch leakage controls
Segmentation (optional)
Cellpose / SAM-based cell extraction and masking
Representation Learning
CNN / ViT backbone, self-supervised or supervised training
State Prediction
Classification (early/late) or regression (continuous differentiation score)
Evaluation & Explainability
Calibration, saliency maps, feature clustering, biological validation
